# ğŸ“š ReadMe: Smart E-Library Portal & Management System

**ReadMe** is a smart e-library portal and management system developed as a finals project for our Object-Oriented Programming (OOP) subject (Group 4: The Classified). It allows users to read eBooks online, receive book recommendations, and provides an admin dashboard for managing library content and users. This project demonstrates real-world application of OOP concepts, CRUD operations, MVC pattern, and full-stack development for a digital library system.  

---

## âœ¨ Features

- **User Portal**
  - Register and log in as a member
  - Browse/search books by category, author, or title
  - Read eBooks directly in the browser
  - Get personalized book recommendations
  - View book details, including cover, author, description, and ISBN

- **Admin Dashboard**
  - Overview of total users, books, and reads
  - Daily reads and category distribution analytics (Charts)
  - Manage book files and metadata
  - Manage users (add, remove, update)
  - View most popular books

- **Tech Stack**
  - **Backend:** ASP.NET (C#), SQLite
  - **Frontend:** Razor Pages, Bootstrap, CSS, JavaScript
